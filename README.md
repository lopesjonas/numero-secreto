# Gerador de Tabuadas e Operações Matemáticas

Um programa Python interativo para gerar tabuadas e realizar operações matemáticas simples.

## Funcionalidades

- **Multiplicação**: Gera tabuada de multiplicação
- **Adição**: Gera tabuada de adição
- **Subtração**: Gera tabuada de subtração
- **Divisão**: Gera tabuada de divisão com tratamento de divisão por zero

## Requisitos

- Python 3.14+

## Como usar

```bash
python test.py
```
